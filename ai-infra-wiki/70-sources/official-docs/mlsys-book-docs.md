# MLSys Book Docs

## Why It Matters

MLSys Book 是把模型、系统、硬件、性能和工程实践串成一张图的高质量总纲，适合给 AI Infra Wiki 提供统一术语和全局心智模型。

## Key Claims

- 它覆盖了训练、推理、数据、硬件、编译和系统优化等多个层次。
- 适合作为学习路线和概念索引的一级来源，而不是某个具体框架 API 的替代品。
- 当需要解释系统全景、性能权衡和术语关系时，本书比单个框架文档更适合做总入口。

## Limits Or Caveats

- 它是系统化教材，不是实时更新的产品文档。
- 某些框架细节和最新实现可能落后于当前版本。
- API、配置和运行时默认行为仍应回到框架官方文档或代码。
